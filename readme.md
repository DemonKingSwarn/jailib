# jailib

just a game being written using raylib and jai

to build the game you must have Jai version:

```sh
Version: beta 0.2.026, built on 21 February 2026.
```

just run:

```sh
jai main.jai
```

to build the game

> [!NOTE]
>
> you need to have raylib for this project to work, download the raylib zip file and extract it and keep its directory here
